# Desafio-Classificador de nível de heróis
1° Projeto do curso 'Blip-Lógica de Programação' da DIO, com o objetivo de fixar o conteúdo e aplicar de uma forma prática e divertida, entendendo, assim, como se usa estruturas lógicas, variáveis, laços de repetição e tudo o que foi ensinado durante o curso até o atual momento.
